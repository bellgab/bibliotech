# Bibliotech
Egy könyvtári menedzser, amely lehetővé teszi a könyvek és olvasók nyilvántartását.


## Nyelvezet
- Felhasználói felület: magyar
- Fejlesztői dokumentáció: magyar
- Kód: magyar

## Fejlesztői környezet
- PHP 8.0+
- Laravel 8+
- MariaDB 10.5+
- NGINX 1.18+
- Docker környezet:
1. NGINX
2. PHP-FPM
3. MariaDB
